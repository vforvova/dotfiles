# Vova's Dotfiles

## Atom
**In use**

1. Install Atom with `brew`.
   ```bash
     brew cask install atom
   ```

2. Update `.atom` using `stow`.
   ```bash
     cd ~/dotfiles
     stow -S atom
   ```

3. Packages are managed via `apm`. Download and install starred packages with `apm starred --install`
